# Zero Net Export 0.4.16 Release Validation

Date: 2026-07-17

## Scope

Release `v0.4.16` delivers `ZNE-FR-018`, the Managed Devices Fleet List on/off
traffic-light indicator.

The release adds a `Power` column to each managed row. The column renders a
green `On` or red `Off` indicator from the existing `observed_active` runtime
state, with text and accessible labels so color is not the only signal.
