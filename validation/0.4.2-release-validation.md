# Zero Net Export 0.4.2 Release Validation

Date: 2026-07-08
Status: released and live-validated

## Scope

Release `0.4.2` covers `ZNE-FR-011` and `ZNE-FR-012`:

- Make the Overview Reconciliation Status card feel more realtime.
- Show last-updated/freshness context for live reconciliation values.
- Fill Source Power, Battery Power, and Confidence from current runtime/source state.
- Explain stale/missing source blockers instead of showing blank or unit-only rows.

Implementation evidence:

- `custom_components/zero_net_export/frontend/zero-net-export-app.js`
- `validation/zne-fr-011-012-overview-console-live-metrics.md`
