# Zero Net Export 0.2.0 release validation

Date: 2026-06-26
Status: GitHub-published; HACS/Home Assistant live validation complete

## Scope

Release `0.2.0` is the first Home Assistant application candidate for Zero Net
Export. It adds a sidebar-default `Zero Net Export` app shell using Home
Assistant `2026.6.4` supported panel/static-asset paths and keeps the existing
integration backend as the control engine.

## Implemented

- Integration version bumped to `0.2.0`.
- HACS minimum Home Assistant version raised to `2026.6.4`.
- Static app module added at
  `custom_components/zero_net_export/frontend/zero-net-export-app.js`.
- App panel bootstrap helper added at
  `custom_components/zero_net_export/app_api.py`.
- Integration setup now registers frontend assets with `StaticPathConfig` and
  the sidebar app with `panel_custom.async_register_panel(...)`.
- App shell renders a nonblank Overview/Sources/Managed Devices/Controls/
  Runtime/Diagnostics layout and a backend readiness/version value.
- Docs and tests updated for GitHub/HACS-only `0.2.0` application delivery.
