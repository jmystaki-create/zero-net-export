# Zero Net Export 0.1.102 corrective release validation

Date: 2026-05-01
Status: repo-validated; publish/live validation pending.

## Scope included

Corrective release for the `0.1.101` ZNE-FR-007 release miss.

- Adds visible Tier 1 native Home Assistant device-page launcher buttons:
  - Open Sensors setup
  - Open Controls setup
  - Open Managed Devices setup
  - Open Diagnostics setup
- Each Tier 1 launcher points to the corresponding Tier 2 native Home Assistant Configure/options flow section and creates a native persistent notification with the exact path when pressed.
- Curates the primary Tier 1 Sensors card to the accepted summary set:
  - Status
  - Health status
  - Active controlled power
  - Battery state of charge
  - Battery reserve SOC
  - Actions today
- Moves detailed release/source/managed-fleet/action-history/planning telemetry into Diagnostics so the main native device page is no longer a dense command wall.
- Preserves existing Diagnostic and Connected Devices surfaces.
