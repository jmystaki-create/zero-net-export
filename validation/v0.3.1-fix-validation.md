# v0.3.1-fix Release Validation

**Date**: 2026-07-07  
**Status**: READY FOR LIVE VALIDATION

## Summary

Corrected the v0.3.1 release where `manifest.json` still showed version `0.3.0`, causing Home Assistant/HACS to report the wrong version.

## Problem

- v0.3.1 GitHub release was created on 2026-07-05 with all Milestone 6 features
- However, `manifest.json` still contained `"version": "0.3.0"`
- When HACS installed v0.3.1, it read the manifest and reported version 0.3.0
- User reported: "3.1 is deployed, but the UI is showing 3.0"

## Fix Applied

1. Updated `manifest.json` version from `0.3.0` to `0.3.1`
2. Committed as `8c3fa26`: "fix: update manifest version to 0.3.1 for v0.3.1 release"
3. Created tag `v0.3.1-fix` pointing to the corrected commit
4. Published GitHub Release v0.3.1-fix
5. Updated CHANGELOG.md with v0.3.1-fix entry
6. Updated PROJECT_STATUS.md to reflect released status
